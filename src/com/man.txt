NAME
	com2pgsql
	
SYNOPSIS
	com2pgsql [OPTION]
	
DESCRIPTION
	Create a PostgreSQL database which contains information regarding aspects of social connectivity over
	a technical project.
	
	-b NAME
		insert Bugzilla communication information from database NAME
		
	-d NAME
		specify the database name which to create and dump to
		
	-e FILE
		insert email communication information from the archive FILE
